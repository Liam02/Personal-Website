Example Trading Strategy using Renko bars for TradingView

This is an example trading strategy I've made on the charting/trading platform "TradingView". The strategy can actually be made into a trading bot if you use the automated trading platform "TradersPost". TradersPost connects the TradingView platform with your broker to make trading automated. The proscess goes somewhat like this; TradingView sends a post request to TradersPost, TradersPost receives the request and decides what signal to send to your broker, Sell, Buy, Exit, etc, then your broker receives the signal and follows the instructions on what to do.

Health Device Simulator using Qt C++

This is a health device simulator, specifically the Radotech Health device. Essentially, the device can scan a users health metrics to help a user identify what they need to work on to improve there health. The health metrics are saved on every scan so a user can go back and look at their previous metrics to see their progress.

I collaborated with 4 other colleagues to make this device simulator. Our task were to make use cases, class diagrams, sequence diagrams, state machine diagrams, then complete the code using our diagrams as reference.
